## Three-Tier Architecture

The writer routes to one of three build functions based on tier:

```python
tier = int(batch_line.split(':')[3]) if len(batch_line.split(':')) > 3 else 1
if tier == 1:
    html = build_html_t1(slug, term, category, p1, content_html, pub_date, quotes_filtered)
elif tier == 2:
    html = build_html_t2(slug, term, category, p1, content_html, pub_date, quotes_filtered)

    html = build_html_t3(slug, term, category, p1, content_html, pub_date)
```
