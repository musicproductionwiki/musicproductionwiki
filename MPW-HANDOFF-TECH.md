```

nav.bible-main-nav  top:0 (media query override)
.bible-mobile-bar   position:sticky  top:60px z:300    height:36px
.entry-nav          top:96px (60+36)
```

### Critical CSS Rule
`overflow:clip` on html/body — NOT overflow:hidden. overflow:hidden creates a new stacking context that kills position:sticky on all descendants. overflow:clip blocks horizontal overflow without creating stacking context.

### Nav Z-Index Hierarchy
- #reading-progress: 99999 (fixed, always on top)
- Dropdowns: 99999 (above entry nav)
